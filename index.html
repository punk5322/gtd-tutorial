Step 1

📝 Capture

Get it out of your head. Don't organize while capturing.

You say:

"capture Finish the Q2 deck"

Sam does:

→ Adds to Inbox.md
→ Logs to PostgreSQL
→ "Captured: Finish the Q2 deck"
Step 2

🔄 Process Inbox

For each item in Inbox, decide what it means:

Schedule
→ Next Actions + date
Someday
→ Someday Maybe
Trash
→ Delete it
Now
→ Priority task
Step 3

📅 Context Views

Filter tasks by where you can do them:

@calls

Phone calls (car, walking)

@computer

Laptop/email tasks

@outside

Errands, in-person

@home

Personal/home

@work

Business tasks

@waiting-for

Delegated items
Step 4

🗓️ Tickler System

Defer items to specific dates:

Defer to day

"defer to 20"
→ Tickler/1-31/20.md

Defer to month

"defer to May"
→ Tickler/Jan-Dec/May.md

On the 20th (or 1st of month), Sam shows what's due.

Step 5

📁 Projects

Multi-step outcomes with next actions:

"new project Mobile App"
→ Creates Projects/Mobile-App.md

Sam asks:
- "What's the outcome?"
- "What's the first next action?"
Step 6

📚 Reference Folder

Non-actionable info you need to keep visible:

"add note Cooper meeting notes"
→ Reference/Meeting-Notes/Cooper-meeting-YYYY-MM-DD.md

Template:
- Date:
- Attendees:
- Key points:
- Action items:
- Follow-up:
Step 7

📊 Weekly Review

Interactive walkthrough (every Friday):

  1. Process all Inbox items
  2. Check projects without next actions
  3. Review Someday/Maybe for激活items
  4. Set top 3 for next week
